Gesture Mouse

This python project uses OpenCV and pynput to move the mouse cursor(and click) on the screen completely gesture controlled. The mouse can be moved by moving the fist clenched. Opening the fist will result in a left click.
For this project I made a HAAR classifier using OpenCV Machine Learning.

Secure Elections

This project is a website to host elections. It maintains the anonimity of the voter from all the other voters and candidates. The data integrity is preserved through a cryptographic hash check which prevents the votes in the database from being altered while preserving anonimity of the vote.
